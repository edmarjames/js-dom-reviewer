Session 5 - Manipulating element's styles

Using style

Syntax: element.style.property = "style"

Using cssText

Syntax: element.style.cssText = 'color: red; background-color: yellow';

Using setAttribute

Syntax: element.setAttribute('style', 'color: red; background-color: yellow');

Using cssText to append styling

Syntax: element.style.cssText += 'color:red; background-color:yellow';

Using getComputedStyle

Syntax: element.getComputedStyle(element, [,pseudoElement]);

Using className

Syntax: element.className;
Syntax: element.className = 'className';

Using classList

Syntax: element.classList;
Syntax: element.classList.add("className1", "className2");
Syntax: element.classList.remove("className1", "className2", "className3");
Syntax: element.classList.replace('className', 'newClassName');
Syntax: element.classList.contains('className');
Syntax: element.classList.toggle('className');

Using clientWidth and clientHeight

Syntax: element.clientWidth;
Syntax: element.clientHeight;

Using offsetWidth and offsetHeight

Syntax: element.offsetWidth;
Syntax: element.offsetHeight;
